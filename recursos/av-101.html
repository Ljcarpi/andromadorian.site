Videos con IA (FFmpeg + TTS + stock)

Armá un pipeline de video con voz sintética, música con ducking y clips de banco de stock, todo automatizado y listo para publicar en redes o sitios.

Nivel: Inicial
Duración: 6–8 h
Formato: Texto + Código + Video
Actualizado: 2025-09-26
Precio: Gratis

Sobre el curso

Este curso te guía paso a paso para producir videos con IA combinando FFmpeg para edición, TTS para la narración y material de bancos de stock (imágenes y clips). Comenzamos con un proyecto base y lo llevamos hasta un pipeline publicable, incluyendo tips de rendimiento y organización de assets.

Aprenderás a crear intros/outros, a sincronizar audio y video, a aplicar side-chain compression (ducking) para que la voz destaque, y a exportar versiones optimizadas para YouTube, Instagram o TikTok.

Requisitos

  • • Computadora con Windows, Linux o macOS.
  • • FFmpeg instalado y accesible por terminal.
  • • Editor de código (VS Code recomendado).
  • • Cuenta en un proveedor de TTS (p.ej., ElevenLabs o similar) o TTS local.
  • • Acceso a bancos de stock (Pexels/Unsplash/otros) o material propio.
  • • Conocimientos básicos de línea de comandos.

¿Qué podré hacer?

  • ✓ Generar narraciones con TTS y mezclarlas con música y efectos.
  • ✓ Sincronizar imágenes/clips de stock con la voz para crear ritmo visual.
  • ✓ Aplicar ducking para que la voz se escuche clara sobre la música.
  • ✓ Automatizar pasos repetitivos con scripts y plantillas.
  • ✓ Exportar en formatos y relaciones de aspecto para distintas plataformas.
  • ✓ Organizar un pipeline reproducible y escalable para múltiples videos.
